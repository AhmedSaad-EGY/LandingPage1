# Landing Page Project

A modern, responsive landing page built as part of my frontend development journey. This project focuses on clean UI design, responsive layouts, and optimized web performance.

## 🚀 Technologies Used
* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom styling, Flexbox, and Grid for a modern layout.
* **Responsive Design:** Fully compatible with mobile, tablet, and desktop screens.

* `/html`: Contains the main structure of the page.
* `/css`: Includes custom stylesheets.
